배열
===

형식 : 자료형 [] 배열참조변수 = new 자료형[갯수];

### 선언 참조변수만 선언
```
int[] arr;
```

### 선언과 동시에 배열 크기 할당
```
int[] arr = new int[5];
String[] arr = new String[5];
```


### 기존 배열의 참조 변수에 초기화 할당하기
```
int[] arr;
arr = new int[]
```

### 선언과 동시에 배열의 크기 지정 및 초기화
```
int[] arr = {1,2,3,4,5};
int[] arr = new int[] {1,2,3,4,5};
int[] arr = {1,2,3,4,5}
String[] weeks = {"월", "화", "수", "목", "금"};
```

### 2차원 배열 선언
```
int[][] arr = new int[4][3];
int[][] arr2 = {{1,2,3,}, {4,5,6}, {7,8,9}}
```

```
arr2[0] -> {1,2,3}
arr2[1] -> {4,5,6}
arr2[2] -> {7,8,9}
```
